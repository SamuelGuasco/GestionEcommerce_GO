# Sistema de Gestión de E-commerce

## Descripción del proyecto
Este proyecto corresponde al desarrollo de un **Sistema de Gestión de E-commerce**, realizado como parte de la materia **Programación Orientada a Objetos**, utilizando el lenguaje de programación **Go** y aplicando principios de **programación funcional**.

El proyecto se desarrolla de forma **individual**, cumpliendo con las disposiciones de trabajo establecidas en la asignatura.

## Objetivo del sistema
Desarrollar un sistema que permita administrar productos, usuarios y pedidos, aplicando estructuras de datos y funciones vistas en la **Unidad 1**, fortaleciendo el aprendizaje de la programación orientada a objetos en Go.

## Módulos del sistema
- Gestión de Productos  
- Gestión de Usuarios  
- Gestión de Pedidos  
- Control básico de inventario  

## Estructura del sistema
El sistema estará organizado mediante funciones y estructuras de datos, siguiendo un enfoque de **programación funcional**, con una separación lógica de responsabilidades para facilitar su mantenimiento y escalabilidad.

## Paquetes utilizados
- `fmt`: Entrada y salida por consola  
- `bufio` y `os`: investigados como herramientas para la lectura de entradas del usuario desde consola, lo cual será útil para la interacción con el sistema en etapas posteriores.
- `strings`: utilizado para limpiar y normalizar texto ingresado por el usuario.
- `strconv`: utilizado para convertir datos de tipo texto a valores numéricos.
- `time`: considerado para el manejo de fechas y marcas de tiempo en pedidos u órdenes. 

*No se utilizan paquetes de terceros en esta etapa del proyecto.*

## Alcance del proyecto
El proyecto contempla:
- Planeación del sistema
- Definición de módulos
- Organización del código en paquetes
- Uso de Git y GitHub para control de versiones

## Tecnologías utilizadas
- Lenguaje: Go  
- Control de versiones: Git  
- Repositorio: GitHub  
- Entorno de desarrollo: Visual Studio Code
